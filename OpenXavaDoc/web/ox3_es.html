OpenXava 3.0 está disponible para descargar en estado beta.
OpenXava 3 permite desarrollar aplicaciones definiendo componentes de negocio con entidades EJB3, es decir con clases de Java convencionales y anotaciones.
Por ejemplo, con solo esta clase:
@Entity
public class Profesor {
  
	@Id @Column(length=5) @Required   
	private String id;

	@Column(length=40) @Required
	private String nombre;


	// Aquí los setters y getters ...
}
Obtenemos una aplicación como esta. Sólo necesitamos escribir una clase Java.
Aunque beta, OX3 tienes todas las característica de OX2, estás invitado a probarlo y a desarrollar tus aplicaciones con él.
