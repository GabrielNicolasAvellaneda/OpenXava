Aquí podemos encontrar algunas aplicaciones demo desarrolladas con OpenXava:
  • MiEscuela: Se usa como base para el tutorial. Realmente simple.
  • Seam Booking: Una versión OpenXava de una demo del marco de trabajo Seam de JBoss.
  • Producto: Implementación OpenXava de un UML Enterprise Component del libro Java Modeling In Color With UML (Coad, Lefebvre, De Luca).
  • OpenXavaTest: Aplicación usada para probar todas las características de OpenXava (para ejecutar la suite completa de pruebas junit).
Todo el código para estas demos está incluido en la distribución de OpenXava.
Es una pena, pero las aplicaciones que de verdad muestra toda la potencia de OpenXava son aplicaciones comerciales y no pueden ser incluidas aquí.

Si quieres contribuir a OpenXava, crear una aplicación demo y contribuirla al proyecto puede ser una buena forma de hacerlo. Si estás interesado pon tu sugerencia en el foro.
