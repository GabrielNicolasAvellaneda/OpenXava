OpenXava 3.0 is now available for download in beta status.
OpenXava 3 allows you to develop applications defining business component using EJB3 Entities, that is using Plain Java Classes and annotations.
For example, with only this class:
@Entity
public class Teacher {
  
	@Id @Column(length=5) @Required   
	private String id;

	@Column(length=40) @Required
	private String name;


	// Here setters and getters ...
}
You will obtain an application like this one. You only need to write one Java class.
Although beta, OX3 has all features of OX2, you are invited to test it, and to develop your applications with it.
