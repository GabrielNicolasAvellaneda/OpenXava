If you are new to OpenXava the better is downloading the OpenXava Core and also the OpenXava Portal, in this way you will have OpenXava code, all demos, documentation and a portal server ready to use for the tutorials and demos.

In order to start you only need to decompress OpenXava Core and OpenXava Portal and follow the instructions in readme.txt file in the openxava-2.2 folder.

OpenXava Core

Contains OpenXava code, the code for all demos, the reference guide, some tutorials, API doc, etc.
  • openxava-2.2.zip
  • openxava-2.2.tar.gz

OpenXava Portal

A portal configured with all demos and database for tutorials. Very useful to start quickly with OpenXava. It's powered by Tomcat and Liferay:
  • openxava-2.2-portal.zip
  • openxava-2.2-portal.tar.gz

Reference Guide

Using these links you'll download the reference guide from SourceForge:
  • openxava-2.2.5-reference-guide.pdf.zip
  • openxava-2.2.5-guia-referencia.pdf.zip

Updating

If you want to update your OpenXava from version 2.x.x to 2.2.5 download the next files and uncompress them in your workspace:
  • openxava-2.2.5-update.zip (only the OpenXava project)
  • openxava-2.2.5-update.tar.gz (only the OpenXava project)
  • openxava-test-2.2.5-update.zip (OpenXavaTest project)
  • openxava-test-2.2.5-update.tar.gz (OpenXavaTest project)
In the other hand if you want to update an old version of OpenXava to the 2.2 version download the next files (that contain OpenXava, OpenXavaTest, AccessTracking and template projects) and uncompress them in your workspace:
  • openxava-2.2-for-updating.zip
  • openxava-2.2-for-updating.tar.gz

OpenXava 3 beta

Try out the new OpenXava:
  • openxava-3.0beta5.zip

Eclipse Plugins

This Eclipse plugin includes a database reverse engineering utility for creating OpenXava components from existing database tables, a wizard for creating new projects (v2.x) and several editor for OpenXava files. In order to install you only need to uncompress it inside the Eclipse 'plugins' folder.
  • openxava-eclipse-plugins-0.2.zip

Maybe you prefer to go directly to the SourceForge download page.
