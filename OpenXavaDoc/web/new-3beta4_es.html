@SearchKey

Ahora es posible marcar una propiedad como clave búsqueda. Las propiedades de búsqueda se usarán por el usuario para buscar en la interfaz de usuario por referencias.
Es decir, podemos escribir un código como esto para una Factura por ejemplo:
    @Id @GeneratedValue(generator="system-uuid") @Hidden 
    @GenericGenerator(name="system-uuid", strategy = "uuid")
    private String oid;

    @SearchKey
    @Column(length=4) @DefaultValueCalculator(CurrentYearCalculator.class)
    private int año;
    
    @SearchKey
    @Column(length=6)
    private int numero;
Y cuando el usuario busque una Factura desde otra entidad, el usuario podrá teclear el año y el número para poder buscarla.
