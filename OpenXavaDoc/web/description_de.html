OpenXava ist ein Java-XML Framework der das Erstellen von Geschaftsapplikationen einfach macht.

Seine Starke besteht darin das XML das Herzstuck ist und nicht Java.

Merkmale:
  • Seit Jahren in realen Applikationen eingesetzt.
  • Konzentiert auf Einfachheit und Produktivitat.
  • Kurze Lernkurve.
  • Flexibilitat. Bietet die Moglichkeit unsere eigenen Funktionen an verschiedenen Punkten einzubringen.
  • Grundlage bildet das Konzept der Geschaftskomponenten.
  • Vorstandig MVC.
  • Grafisches Webinterface mit allen typischen Merkmalen einer Geschaftsapplikation.
  • Angepast um mit Legacy Datenbanken zu arbeiten.
  • Erstellt eine komplete J2EE Applikation (inklusive EJB).
  • Die Grundlage bilden Standards wie XML, J2EE, EJB, JSP, etc.
  • Ist OpenSource (LGPL) und benutzt OpenSource Tools wie Ant, JUnit, JasperReports, TL, XDocLet, Hibernate, etc.
  • Der Entwickler kann sich zwischen Englisch und Spanisch entscheiden.
  • Alle Etiketten und Meldungen erscheinen in Englisch, Spanisch y Katalanisch; Im Moment...
