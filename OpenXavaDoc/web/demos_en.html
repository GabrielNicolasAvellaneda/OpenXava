Here you can find some demo applications developed using OpenXava:
  • MySchool: It's used as base for the tutorial. It's really simple.
  • Seam Booking: An OpenXava version of a demo from the JBoss Seam framework.
  • Product: OpenXava implementation of a UML Enterprise Component from the book Java Modeling In Color With UML (Coad, Lefebvre, De Luca).
  • OpenXavaTest: Application used to test all OpenXava features (to run full junit suite).
All the code for these demos are included in the OpenXava distribution.
It's a shame, but the applications that really show all power of OpenXava are commercial applications and cannot be included here.

If you want to contribute to OpenXava, creating a demo application and contributing it to the project can be a good way. If you are interested put your suggestion in the forum.
