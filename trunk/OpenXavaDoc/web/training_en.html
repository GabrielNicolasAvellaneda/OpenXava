Gestión 400 organizes on-demand OpenXava training.
To enrol call (34) 96 303 9400 and ask for Juan Vicente Llavador.

The language for courses is Spanish and the place is Valencia (Spain). If you are interested in other language, location or course content please contact with Javier Paniza by email (the email is in the Reference Guide).

Basic course: OpenXava 2.x

The core of course will be the development of a little application using OpenXava, using this example we'll  learn:
  • Introduction to J2EE/JavaEE and OpenXava.
  • Development of CRUD and automatic reports.
  • References and colections.
  • Calculated properties.
  • User Interface customization.
  • Deploy on Liferay Portal.
  • Events.
  • Refining the behaviour of a controller.
  • Creation of tab filters para el tab.
  • Creation a custom controller.
  • Methods.
  • Writing custom logic using POJOs and Hibernate.
Laptop required (512 Mb minimum, 1 Gb recommend).
The course duration is 16 hours.

Advanced course: OpenXava 2.x

In this course we continue developing the application of the basic source. We'll learn:
  • JUnit automatic testing for OpenXava applications.
  • Many-to-many associations.
  • Custom logic using EJB3.
  • Custom editor and stereotype.
  • Using custom report designed with iReport/JasperReport.
  • Etc
Laptop required (512 Mb minimum, 1 Gb recommend).
The course duration is 12 hours.
