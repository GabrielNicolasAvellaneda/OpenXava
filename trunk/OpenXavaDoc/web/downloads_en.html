If you are new to OpenXava the better is downloading the OpenXava Core and also the OpenXava Portal, in this way you will have OpenXava code, all demos, documentation and a portal server ready to use for the tutorials and demos.

In order to start you only need to decompress OpenXava Core and OpenXava Portal and follow the instructions in readme.txt file in the openxava-2.1 folder.

OpenXava Core

Contains OpenXava code, the code for all demos, the reference guide, some tutorials, API doc, etc.
  • openxava-2.1beta.zip
  • openxava-2.1beta.tar.gz

OpenXava Portal

A portal configured with all demos and database for tutorials. Very useful to start quickly with OpenXava. It's powered by Tomcat and Liferay:
  • openxava-2.1beta-portal.zip
  • openxava-2.1beta-portal.tar.gz

Reference Guide

Using these links you'll download the reference guide from SourceForge:
  • openxava-2.1beta-reference-guide.pdf
  • openxava-2.1beta-guia-referencia.pdf

Updating

This distribution is for updating your current version of OpenXava to the 2.1. It only contains the code for the  OpenXava, OpenXavaTest, AccessTracking and template projects:
  • openxava-2.1beta-for-updating.zip
  • openxava-2.1beta-for-updating.tar.gz


Maybe you prefer to go directly to the SourceForge download page.
