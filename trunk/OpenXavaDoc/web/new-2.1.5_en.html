New stereotype IMAGE_LABEL

It shows an image localized in /xava/images of your web application.
You can declare a property in this way:
<property name="sample" stereotype="IMAGE_LABEL"/>

If the value of the property is 'RED' then the image /xava/images/imageLabel_RED.gif will be shown; if the value of the property is 'BLACK' then the image /xava/images/imageLabel_BLACK.gif, and so on.

May be useful, for example, for showing an image associated to each value of a <valid-values/>.
