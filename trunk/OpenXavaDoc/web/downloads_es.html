Si eres nuevo con OpenXava lo mejor es descargar el Nucleo de OpenXava y también el Portal OpenXava, de esta manera tendrás el código de OpenXava, todas las demos, documentación y un portal listo para usar para los tutoriales y las demos.

Para empezar solo necesitas descomprimir OpenXava (nucleo) y OpenXava Portal y seguir las instrucciones en el archivo leeme.txt en la carpeta openxava-2.1.

Nucleo de OpenXava

Contiene el código de OpenXava code, el código de todas las demos, la guía de referencia, algunos tutoriales, la doc del API, etc:
  • openxava-2.1.zip
  • openxava-2.1.tar.gz

Portal de OpenXava

Un portal configurado con todas las demos y las bases de datos para los tutoriales. Muy útil para empezar rápidamente con OpenXava. Es un Tomcat + Liferay:
  • openxava-2.1-portal.zip
  • openxava-2.1-portal.tar.gz

Guía de referencia

Usando estos vínculos descargarás la guía de referencia desde SourceForge:
  • openxava-2.1-reference-guide.pdf.zip
  • openxava-2.1-guia-referencia.pdf.zip

Actualización

Esta distribución sirve para actualizar tu versión actual de OpenXava a la 2.1. Solo contiene el código de los proyectos OpenXava, OpenXavaTest, AccessTracking y los proyectos plantilla:
  • openxava-2.1-for-updating.zip
  • openxava-2.1-for-updating.tar.gz

Eclipse Plugins

Este plugin de Eclipse es una utilidad de ingenieria inversa para bases de datos que permite crear componentes OpenXava a partir de tablas de base de datos existentes. Para instalarlo solo es necesario descomprimirlo dentro de la carpeta del Eclipse.
  • openxava-eclipse-plugins-0.1.zip

Quizás prefieras ir directamente a la página de descargas de SourceForge.
