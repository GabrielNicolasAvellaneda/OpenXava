OpenXava es un marco de trabajo que con XML y Java permite desarrollar fácilmente aplicaciones de gestión.

Su virtud reside en que el corazón de nuestras aplicaciones es XML no Java.

Características:
  • Usado durante años para hacer aplicaciones reales.
  • Centrado en conseguir sencillez y productividad.
  • Curva de aprendizaje corta.
  • Flexible. Es posible insertar nuestra propia funcionalidad en cualquier punto.
  • Basado en el concepto de componente de negocio.
  • Totalmente MVC.
  • Interfaz gráfico web con rica funcionalidad propia de aplicaciones de gestión.
  • Adaptado para trabajar con esquemas de base de datos legados.
  • Generá una aplicación J2EE (incluyendo EJB) completa.
  • Basado en estándares como XML, J2EE, EJB, JSP, etc.
  • De código abierto (LGPL) y basado en otras herramientas de código abierto como Ant, JUnit, JasperReports, TL, XDocLet, Hibernate, etc.
  • El desarrollador puede usar Inglés o Español.
  • Todas las etiquetas y mensajes están en Inglés, Español y Catalán; de momento...
